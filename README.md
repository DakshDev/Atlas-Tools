####  Weather API Responde

* name: `location-name`
* country: `country-name`
* localtime: `localtime`
* temp_c: `temp_c`
* temp_f: `temp_f`
* wind_kph: `wind_kph`
* is_day: `0 - 1` <!-- 0 = night & 1 = day --> 
* condition.text: `sunny, fog, clear etc.`
* humidity: `humidity`
* cloud: `0 - 100`
